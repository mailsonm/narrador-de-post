Deixe seus usuários ouvirem o seu conteúdo com o widget de conversão de texto em fala do narrador-de-post

== Descrição ==

O plugin se comporta como um widget de conversão de texto em voz onde ele é incorporado em suas postagens e oferece aos usuários uma maneira alternativa de consumir seu conteúdo como áudio.

Os artigos são sintetizados apenas na primeira vez. Quando um artigo é atualizado, apenas os parágrafos alterados são ressintetizados.


== Perguntas frequentes ==

# O que é a tecnologia Text to Speech (TTS)?
A tecnologia Text to Speech lê em voz alta o texto digital — as palavras em computadores, smartphones e tablets. O TTS ajuda pessoas com deficiências de aprendizagem a melhorar sua compreensão de leitura.

# Como funciona o widget narrador-de-post?
O widget narrador-de-post aparece no topo de suas postagens como um botão rotulado "Ouvir este artigo". Quando clicado, o botão é substituído por um reprodutor de áudio que reproduz a fala sintetizada do conteúdo do seu artigo.

# Quanto custa?
Nada, porem você pode fazer uma doação via pix para ajudar o desenvolvedor 97165c03-707b-4458-b652-2b124361a663.

# Como os usuários do meu site usam o widget? Meus usuários precisam baixar alguma coisa?
Os usuários não precisam baixar ou instalar nada para usar o widget. Quando eles clicam no botão "Ouvir este artigo" no seu site, o widget será incorporado diretamente na página para permitir a reprodução de fala sintetizada.

# Quais dispositivos e navegadores suportam o widget narrador-de-post?
O widget funciona em todos os tipos de dispositivos, incluindo desktops, celulares e tablets. Ele suporta todos os navegadores modernos.

# Quais idiomas o widget narrador-de-post suporta?
O widget suporta está configurado para pt-BR, mas pode ser ajustados para outros idiomas.

# Que tipo de dados e relatórios o widget coleta e compartilha?
O widget coleta a URL do artigo ouvido, hora do dia, tipo de dispositivo (computador/celular), sistema operacional e GEO.

# Onde você recomenda que um webmaster coloque o botão Ouvir em um site?
Recomendamos colocar o widget no topo dos artigos ou em um local visível acima da dobra.

# Vocês vendem os dados do usuário do meu site?
Não, não vendemos dados de clientes ou usuários a terceiros.

# O widget narrador-de-post suporta DOCs e PDFs?
O widget não suporta a leitura de DOCs, PDFs ou qualquer tipo de documento que esteja incorporado na página da web.

# Como faço para cancelar?
Você pode cancelar a qualquer momento removendo o widget do seu site. Ao remover o widget do seu site, você não incorre mais em taxas de serviço.
